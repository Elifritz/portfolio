Street Lighting and Crime in Downtown Madison,WI

A capstone project that involved using GIS techniques and spatial statistics to analyze the effects of lighting on petty crimes in downtown Madison, Wi.
Spatial Database Design of Road Density and Deer Population Density in Wisconsin

A database project using SQL and database design to assess how road density, deer population density and deer-related car accidents correlate. The project is contained within a zip file that includes images, papers, slideshows, shapefiles and a database backup file.
Surveying Bascom Hill for Construction of Moving Sidewalk

A GPS-based project that utilizes MATLAB, Garmin receivers, GPX Master and Google Earth to demonstrate the possibility of construction on Bascom Hill in the University of Wisconsin-Madison. In addition, it draws conclusions on whether a phone app can replace a true GPS device. This project is contained within a zip file that includes a paper and a presentation.
